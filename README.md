# ai-machine-learning-application

## python application
- py -m  pipreqs.pipreqs --force .
- py -3 -m venv --system-site-packages venv
- source venv/Scripts/activate
- flask --app main run

## react application
- npm run start

## docker application
- pip3 install -r requirements.txt
- docker run -p 5000:5000 -d main
